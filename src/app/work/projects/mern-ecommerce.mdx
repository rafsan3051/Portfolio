## Overview

A comprehensive e-commerce solution built from scratch using the MERN stack. This platform provides all the essential features needed for running an online store, from product catalog to payment processing.

## Key Features

- **Product Management**: Complete CRUD operations for products with image uploads
- **Shopping Cart**: Persistent cart functionality with quantity management
- **User Authentication**: Secure registration and login system
- **Order Management**: Track orders from placement to delivery
- **Payment Integration**: Secure payment processing
- **Admin Dashboard**: Comprehensive admin panel for managing products, orders, and users
- **Product Reviews**: Customer review and rating system
- **Search & Filter**: Advanced product search with multiple filters

## Technical Stack

### Frontend
- **React.js**: Dynamic and responsive user interface
- **Redux Toolkit**: Efficient state management
- **React Bootstrap**: Responsive design components
- **Axios**: HTTP client for API requests

### Backend
- **Node.js & Express.js**: Robust API server
- **MongoDB & Mongoose**: Database and ODM
- **JWT & bcrypt**: Authentication and encryption
- **Cloudinary**: Image hosting and optimization

## Business Logic

The platform implements a complete e-commerce workflow including:
- User registration and authentication
- Product browsing and searching
- Cart management with session persistence
- Checkout process with address management
- Order tracking and status updates
- Admin controls for inventory management

## Security Features

- Password encryption using bcrypt
- JWT-based authentication
- Protected routes and API endpoints
- Input validation and sanitization
- Secure payment processing
